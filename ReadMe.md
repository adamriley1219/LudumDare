
## DevConcole Input

```
- **~** - Turn Open Console
- **`** - Turn Open Fading Console
- **ESC** - Turn Off Console if open, Closes App if Console already closed.

```


## Debugging Controls
```
- **H** - help
- **W** - Run test Event in console
- **ESC** - to exit.

```
